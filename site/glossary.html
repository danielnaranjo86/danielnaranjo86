Glossary

  1. HTML attributes:

    May also have attributes, which contain information about the element beyond the content and also can be used to identify an element for styling and in scripts.

  2. Docttype:

    👍 Required for modern web sites
    👍 Tells browser how to interpret HTML
    👍 Formerly a way to indicate whichn HTML version the document was in

  3. UTF-8:

    Allows you to use symbols from almost any language, including emoji 😎

  4. Head:

    🔥 There must be only one <head> element per HTML document
    🔥 Contains meta information about the document such as page title, description, and links to stylesheets and scripts

  5. Nested list:

    This is a nested list! 😎✌
    List and list item elements are block-level, so they may contain other block-level elements—including other lists!
